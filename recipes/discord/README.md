# Discord for Franz
This is the official Franz recipe for Discord

## How to install
You can install it manually by following the instructions below.

1. Download this recipe: https://github.com/transnat/recipe-discord/archive/master.zip
2. Open the Franz Plugins folder on your machine:
  * Mac: `~/Library/Application Support/Franz/recipes/dev/`
  * Windows: `%appdata%/Franz/recipes/dev/`
3. Create the `dev` folder if you don't yet have it.
4. Unzip the file you have just downloaded.
5. Copy the `recipe-discord-master` folder into the plugins dev directory
6. Reload Franz
7. Open Franz settings. Under 'Add New Services' you will find a 'Custom Services' section, click on 'Discord' and log in.
